# What is a data structure?

Data structures are organizational tools that allow for more advanced algorithms.

Simple data structures:
  * Lists - ordered collections of data
  * Dictionaries - Key : Value mappings
  * Sets - Unordered collections of unique data

# What is an ['algorithm'](https://en.wikipedia.org/wiki/Algorithm)?

Just a set of instructions that can be carried out to solve a problem. In basic terms, nearly every function written in code is an algorithm.

In the context of computer scidence, an algorithm is a finite sequence of well-defined, computer implementable instructions.
An algorithm must be 3 things:

  * Defined - There is a sequence of steps that perform a task.
  * Unambiguous - There is a "correct" & "incorrect" interpretation of the steps.
  * Implementable - Can be executed using software & hardware
